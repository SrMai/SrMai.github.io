Crazy Games GDL

Crazy Games es una página de ventas online con servicio de reparaciones y chips para consolas, esta pagina fue promocionada en Guadalajara para un local en San Juan de Dios.

Este sitio web cuenta con metodos de pago por paypal y envios nacionales.

Descripcion

Este local tenia bastantes clientes al día, sus ventas atravez la Facebook eran altas pero el tiempo no alcanzaba.

Me di cuenta que si se creaba un mejor sistema de ventas estas aumentaria y hasta se venderia sola la mercancia, asi que comenze a hacer la pagina.

El Problema

El negocio tenia muy buenas ganancias, pero muy mala publicidad online.

Perdian muchas ventas debido a que no se tenian metodos de pago online, servicio a domicilio ni pagina para ver sus productos.

Solución

Se comenzo a crear una pagina solo para ver los productos, cuando nos dimos cuenta que tenia muchas visitas dicha pagina se decidio colocar un metodo de pago y envio solo en el estado de Jalisco.

Cuando comenzaron a subir las ventas localmente y a tener buenas reseñas, se decidio expandirse y hacer envios nacionales, esto creo nuevos empleos en dicho local y se lograron abrir 2 locales mas para cubrir las ventas..

Mejoras

Despues de un tiempo se añadieron mejores fotografias, mas productos y la opción de comprar para mayoristas.

Se añadio una garantia online en productos para inspirar mas confianza, el primer retorno era gratis..
